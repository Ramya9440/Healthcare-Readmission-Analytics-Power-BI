# Healthcare-Readmission-Analytics-Power-BI
End-to-end healthcare analytics project analyzing readmissions, cost drivers, and length of stay using SQL, Python, and Power BI.
Project Summary
This project delivers an end-to-end healthcare analytics solution designed to analyze hospital readmissions, cost drivers, and length of stay (LOS).
Using Power BI, SQL, and Python, the dashboard transforms raw healthcare visit data into actionable insights for hospital administrators and decision-makers.
The focus is on identifying high-risk patient cohorts, cost inefficiencies, and operational improvement opportunities.
